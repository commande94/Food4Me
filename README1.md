# 🚀 Setup IA dans VS Code — React / React Native / Node.js

Configuration complète pour intégrer une IA moderne dans Visual Studio Code avec GitHub Copilot.

---

# 📦 Stack utilisée

* Visual Studio Code
* GitHub Copilot
* GitHub Copilot Chat
* Git
* GitHub CLI
* React
* React Native
* Node.js

---

# 🎯 Objectif

Avoir une IA directement dans VS Code capable de :

* générer du code
* corriger des erreurs
* proposer des composants React
* aider sur React Native
* créer des APIs Node.js
* comprendre le projet
* accélérer le développement

# 6️⃣ Connecter GitHub à VS Code

Après installation :

* une fenêtre navigateur s’ouvre
* connecter le compte GitHub

---

# 7️⃣ Vérifier que l’IA fonctionne

Créer un fichier :

```jsx
App.jsx
```

Écrire :

```jsx
function Login
```

Attendre quelques secondes.

➡️ Une suggestion IA apparaît en gris.

Accepter avec :

```txt
Tab
```
